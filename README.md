### Consol size
* Width : 120 px
* Height : 40 px

### Controls
* __&larr;__ : Move the tetromino to the left
* __&rarr;__ : Move the tetromino to the right
* __&darr;__  : Move the tetromino downwards
* __z__ : Rotate the tetromino clockwise
